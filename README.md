# Exploratory-Data-Analysis-Churn-Analysis
Exploration of Data using Python , Matplotlib and Seaborn
The project is of Exploratory data analysis. In this data from a telecom company is used. It has main variable of whuy people have churned the service. The data contains many variables such as Gender, Service, Duration and Many others. 
We firstly imported the data and Libraries which we will use. In the Second step we inspect the data and checked if there are any duplicate or null values. After inspecting the data, we changed the data types of categorical columns. We also looked for the descriptive statistics of the data. In the last part, we explored the data. 
Results from the Data are shown visually and numerically in the notebook.
